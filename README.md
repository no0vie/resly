# Описание

Сервис для эффективного управления ресурсами. Реализует взаимодействие между 3 ролями пользователей: продавец,
покупатель, администратор. 

Продавец поставляет некоторый товар и дает на него талоны. Покупатель талоны обналичивает товаром у продавца.
Администратор контролирует процесс.

Для удобства покупателя ему доступен интерфейс с картой, где отображаются поставщики, у которых он может реализовать
талон. Также при отсутствии необходимости или ошибке, пользователю такой роли доступна сдача талона.

Продавец видит список предоставленных им талонов и может изменять их на свое усмотрение, при этом у покупателя должен
появиться актуальный статус.

Адмнинистратор служит управляющим над продавцами, но также может вмешиться в управление талонами для покупателей.
